What is MunkiReport-PHP?

Munkireport-php is a reporting client for munki.

The previous version of munkireport is still available on googlecode: munkireport-php. This project is a complete rewrite from the previous version, which was a quick-and-dirty port from the original python based munkireport.

The project a work-in-progress, please check for updates regularly.
